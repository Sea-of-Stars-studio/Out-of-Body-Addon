## 使用
>[!Tip]
>打开游戏时应该会立马显示帮助信息，否则Addon并没有正常加载
### 指令
| 指令 | 功能介绍 |
| :----: | :----:  |
| /fca | 打开设置面板,解锁神秘新功能(也可以在其中操作所有指令功能) |
| /fca tp | 指令传送到灵魂出窍位置(需要管理员在设置面板开启) |
| /fca on | 指令开启灵魂出窍 |
| /fca off | 指令关闭灵魂出窍 |
| /fca help | 帮助 |
